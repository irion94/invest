# /ike — Monitor limitów IKE 2026

## Instrukcje

1. Przeczytaj `memory/portfolio.md` — pobierz:
   - Łączna wartość Twojego IKE (sekcja "Twoje IKE")
   - Pozostały limit IKE 2026 (Ty i Żona)
   - Wolne środki na kontach

2. Przeczytaj `memory/strategy.md` — pobierz plan wpłat IKE 2026

3. Oblicz dla każdego konta:
   - Roczny limit IKE = pozostały limit + wpłacone do tej pory
   - Wpłacone YTD = roczny limit - pozostały limit
   - Wykorzystanie = wpłacone / roczny limit × 100%
   - Pozostałe miesiące 2026 = 12 - aktualny miesiąc
   - Optymalny DCA miesięczny = pozostały limit / pozostałe miesiące

4. Wyświetl tabelę:

| | Twoje IKE | IKE Żony | Łącznie |
|---|-----------|----------|---------|
| Roczny limit 2026 | X PLN | X PLN | X PLN |
| Wpłacono YTD | X PLN | X PLN | X PLN |
| Pozostało | X PLN | X PLN | X PLN |
| Wykorzystanie | X% | X% | X% |
| DCA/miesiąc (opt.) | X PLN | X PLN | X PLN |

5. Dodaj komentarz:
   - Który miesiąc jest "deadline" na wpłaty (koniec roku)
   - Czy DCA jest realistyczny przy obecnym tempie
   - Przypomnienie: IKE = 0% podatku Belki przy trzymaniu do 60. roku życia
   - Zaproponuj `/strategia` jeśli plan wpłat wymaga aktualizacji

## Uwagi
- Limity IKE są ustalane co roku przez KNF — aktualne wartości w portfolio.md
- Nie można przekroczyć limitu — XTB blokuje wpłatę
- Wpłaty na IKE Żony wymagają osobnej akcji (inne konto)
